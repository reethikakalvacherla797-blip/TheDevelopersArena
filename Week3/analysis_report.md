# 📊 Week-3 Sales Data Analysis 

## 📌 Project Overview

This project focuses on analyzing a sales dataset using Python and Pandas.
The goal is to clean the data, explore key metrics, and generate meaningful business insights such as total revenue and best-selling products.

This project was completed as part of Week 3 – Introduction to Data Analysis.

---

## 🎯 Objectives

* Load and explore a real-world sales dataset
* Handle missing values and remove duplicate records
* Calculate important sales metrics
* Identify the best-selling product
* Generate a clear and formatted sales report

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Jupyter Notebook

## 🧹 Data Cleaning Steps

* Filled missing values in the `Total_Sales` column using mean values
* Removed duplicate rows
* Verified dataset consistency

---

## 📈 Analysis Performed

The following metrics were calculated:

* Total Revenue
* Average Sales
* Highest Sale
* Lowest Sale
* Best-Selling Product

---

## 📊 Key Insights

* The dataset contains 100 rows and 5 columns.
* Total revenue was calculated from all sales records.
* The best-selling product was identified using group-by operations.
* Data cleaning improved dataset accuracy before analysis.
