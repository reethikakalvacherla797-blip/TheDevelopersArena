# Week-4 Sales Data Analysis Project

## Project Overview
This project analyzes a sales dataset to understand product performance, regional sales distribution, and overall sales trends. The goal is to use Python data analysis tools to extract meaningful insights and visualize the results.

The project uses Python libraries such as **pandas** for data processing and **matplotlib** for creating visualizations.

---

## Setup Instructions
1. Download or clone the repository from GitHub.
2. Ensure all project files remain in the correct folder structure.
3. Place the dataset inside the **data/** folder.
4. The analysis code is available in **main.py** or **analysis.ipynb**.
5. Generated charts and graphs are stored in the **visualizations/** folder.

## Visualizations

The project includes the following visualizations:

- **Line Chart** – Shows sales trends over time
- **Bar Chart** – Displays total sales by product
- **Pie Chart** – Shows sales distribution by region

All visual outputs are saved in the **visualizations/** folder.
