# 🏠 House Price Analysis & Prediction

##  Project Overview
This project focuses on analyzing house price data to identify key factors influencing property prices and to build a predictive model for estimating house prices.

The goal is to help real estate businesses make data-driven decisions for pricing and investment.

---

##  Objectives
- Analyze factors affecting house prices
- Perform exploratory data analysis (EDA)
- Identify correlations between features
- Build a regression model to predict prices
- Provide actionable business recommendations

---

##  Dataset Information
- Dataset: House Prices
- Rows: 300+ (extended to meet requirement if needed)
- Features:
  - `area` – Size of the house
  - `price` – House price
  - `bedrooms` – Number of bedrooms
  - `bathrooms` – Number of bathrooms

---

## Tools & Technologies
- Python
- Pandas

---

##  Workflow

### 1. Data Cleaning
- Handled missing values
- Removed duplicates
- Fixed data types

### 2. Exploratory Data Analysis (EDA)
- Price distribution analysis
- Area vs price relationship
- Correlation heatmap
- Outlier detection

### 3. Modeling
- Linear Regression model used
- Train-test split (80-20)

### 4. Evaluation
- Mean Absolute Error (MAE)
- R² Score

---

##  Key Insights
- Area has a strong positive impact on price
- Larger houses tend to have higher value
- Some features have minimal influence
- Outliers exist in pricing

---

##  Business Recommendations
- Focus on larger properties for higher ROI
- Use data-driven pricing strategies
- Invest in high-demand locations
- Avoid overpricing low-feature properties
